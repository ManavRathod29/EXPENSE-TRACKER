EXPENSE TRACKER WEB APPLICATION

---

PROJECT OVERVIEW
The Expense Tracker is a simple web-based application that helps users manage their daily expenses. Users can add expenses, categorize them, and view the total spending.

---

OBJECTIVE

* Track daily expenses
* Categorize spending (Food, Travel, Shopping, etc.)
* Display total expenses
* Provide a simple and user-friendly interface

---

FEATURES

* Add new expenses
* Select category for each expense
* View expense list
* Automatic total calculation
* Chart visualization (optional)

---

TECHNOLOGIES USED

* HTML (Structure)
* CSS (Styling)
* JavaScript (Functionality)
* Chart.js (Optional for charts)

---

PROJECT STRUCTURE

expense-tracker/
│
├── index.html
├── style.css
└── script.js

---

HOW TO RUN THE PROJECT

1. Download or create the project folder
2. Save all files inside the folder
3. Open index.html in any web browser
4. Start adding expenses

---

HOW IT WORKS

1. Enter expense name
2. Enter amount
3. Select category
4. Click "Add Expense"
5. Expense will be displayed
6. Total amount updates automatically
